# SaSHA

The modern, secure VM platform and programming language, designed to be a replacement for JADE, VASM, and NexFUSE, and **web-friendly**.

SaSHA is a VM platform and programming language designed to take the best attributes from OpenLUD, NexFUSE, and Jade, while providing abstractions and performant application development not seen in any of those existing formats.

## Modules

SaSHA is *multipart*, in that there is no one way to integrate it into software. SaSHA comes with:

- **A VM Runtime platform**: This runtime platform is designed for performant execution on the web. Fully tested and app-ready.
- **A programming language executor**: The **SaSHA SVL** is designed to be human-readable and modern to the current language standards.
- **VASM Emulator**: SaSHA also comes with a built-in parser for the VASM programming langauage (or, *LunarRED Assembly*)
